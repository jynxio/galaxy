# 概述

基于 [three.js](https://www.npmjs.com/package/three) 的星系，欢迎查看 [在线 demo](https://galaxy-six.vercel.app/)。

![simple](./image-hosting/simple.png)

<br />

# 运行

依次执行下述命令，来执行该项目：

1. `npm install`
2. `npm run start`
